# Photo Sharing Frontend

This is the frontend application for the Photo Sharing project, built with React.

## Running on CodeSandbox

This project is ready to be imported into CodeSandbox. By importing this GitHub repository, CodeSandbox will automatically detect the React environment, install the necessary dependencies, and spin up the development server.

### Scripts

- `npm start`: Starts the development server.
- `npm run build`: Builds the app for production to the `build` folder.
- `npm test`: Launches the test runner in the interactive watch mode.

### API Integration (CORS)

This frontend expects to communicate with the backend API.
If you are running both frontend and backend on CodeSandbox in separate projects, you will need to update the API base URL in your API calls to point to the CodeSandbox URL of your backend project.

You can configure this by setting an environment variable or updating the API requests directly.
